小米 AX3000T 刷 u-boot 后再刷入原厂固件

由于重装系统，丢失了以前备份的 mtd 文件，因此难以刷回原厂系统的 bootloader。

不过在论坛中搜寻到了网友分享的 ubi 文件 1，可以从 u-boot 启动小米原厂固件。

下载链接

只要在 u-boot 的 Web UI 中选择 default 布局，然后上传该 ubi 文件，等待重启后即可恢复为原厂固件。

U-Boot 和 UBI 的关系

在许多嵌入式 Linux 系统中，U-Boot 会首先启动，然后识别和操作 UBI 卷，最终从 UBI 卷中加载并启动 Linux 内核。UBI 提供了一个可靠的底层存储抽象，使得在 NAND Flash 上部署文件系统更加健壮。
