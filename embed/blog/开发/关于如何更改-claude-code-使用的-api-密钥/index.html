关于如何更改 Claude Code 使用的 API 密钥

配置步骤

  1. 编辑配置文件

在 ~/.claude/settings.json 中添加/编辑以包含：

{
	"apiKeyHelper": "~/.claude/anthropic_key.sh"
}
  1. 创建密钥脚本

然后在 ~/.claude/anthropic_key.sh 中：

echo "sk-........."
  1. 设置权限

赋予其可执行权限：

chmod +x ~/.claude/anthropic_key.sh

大功告成！
