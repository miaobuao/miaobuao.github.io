Fish使用nvm(Node Version Manager)

安装fisher

fisher install jorgebucaran/fisher

出现问题：Invalid plugin name or host unavailable: xxx 说明cURL有问题1 参考这篇博客

安装nvm.fish

fisher install jorgebucaran/nvm.fish

安装node版本

# 查看所有版本
nvm list-remote
# 安装最新版
nvm install latest

记住版本

node --version > ~/.nvmrc
echo "nvm use" >> ~/.config/fish/config.fish
