Linux 下玩游戏不能同时使用触摸板和键盘的问题 (GNOME)

最近我又卸载了 Windows， 装回了 Arch Linux。配置好了 NVIDIA 显卡驱动之后正准备爽玩的时候，发现触摸板和键盘不能同时使用，表现为：当使用键盘的时候会中断触摸板的行为。

我是用的是 GNOME 桌面，在终端里输入：

gsettings get org.gnome.desktop.peripherals.touchpad disable-while-typing

发现返回 true，只需要修改为 false 即可：

gsettings set org.gnome.desktop.peripherals.touchpad disable-while-typing false
