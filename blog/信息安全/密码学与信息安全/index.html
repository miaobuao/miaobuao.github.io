对称加密技术

  1. 对称密钥 明文 == key ==> 密文 == key ==> 明文 优点： 速度快 目前主流：
    • DES(美国的IBM设计的): 对明文使用Key，十六轮加密，产生一个密钥 对密钥使用key，十六轮加密，产生明文。 好处： 只需要使用加密芯片就可以解密
    • AES(比利时设计)
  2. 公钥和密钥 明文 == 公钥 ==> 密文 == 私钥 ==> 明文 缺点： 比 “对称密钥” 慢很多
